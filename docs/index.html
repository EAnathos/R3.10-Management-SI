This class is used to manage the connection to the database and the creation of collections. It is used also to create collection, insert data into the collections and drop collections.

new MongoDBManager(databaseName: string, uri: string)
Parameters
databaseName (string) The name of the database to connect to
uri (string) The URI of the database to connect to
Example
const mongoManager = new MongoDBManager("SI_Database", "mongodb://localhost:27017");
